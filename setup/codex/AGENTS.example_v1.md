# Global Agent Instructions

- Read applicable repository instructions and follow the most specific guidance.
- Understand the task and relevant code before making changes.
- Make the smallest complete change and preserve unrelated user work.
- Match existing architecture, style, and conventions.
- Do not perform destructive actions, commit, push, or rewrite history unless explicitly asked.
- Do not manually edit generated files; regenerate them using the appropriate tool when required.
- Resolve minor ambiguity from context and state assumptions. Ask when different interpretations would materially change the result.
- Validate changes in proportion to their risk and report what changed, checks run, and any remaining limitations.
- Keep communication concise and explain consequential decisions or trade-offs.
